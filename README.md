# start

On Command Line (cmd), type:
<code>
git --version
node --version
npm --version

npm install -g yo grunt-cli bower
npm install -g generator-angular 
yo angular:app symbolViewerApp
</code>


<code>
bower list
bower update bootstrap

</code>


# build

<code>
grunt build
grunt serve
grunt buildcontrol:pages
</code>
